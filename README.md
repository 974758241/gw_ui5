# 说明一下该插件如何使用
## 需要在目录下配置一个文件 connectSAP.json
```json
{
    "baseURL":"你服务器的地址和端口",
    "useName":"账号",
    "usePwd":"密码",
    "systemIdentification":"SFD",
    "sapClient" : "100"
}
```
